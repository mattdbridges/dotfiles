# Dotfiles

My dotfiles and system configuration handled by [chezmoi](https://www.chezmoi.io).

## Install

Installing

```
$ sh -c "$(curl -fsLS git.io/chezmoi)" -- init -S $HOME/.dotfiles --apply mdespuits
```
